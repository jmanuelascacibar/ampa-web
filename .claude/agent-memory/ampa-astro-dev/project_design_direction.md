The AMPA website design is intentionally aligned with the school's site (dominicasmadrid.es/comunidad-de-familias/).

**Why:** Manuel wants visual consistency between the school and parent association websites so they feel like part of the same community.

**How to apply:** All new pages, components, and UI elements should use:
- Primary burgundy/wine `#ac3c58` (primary-600) for buttons, headings, brand color
- Lime/chartreuse `#E7FF89` (accent-200) for hover states, decorative highlights, accent pops
- Dark charcoal `#3C3950` (body) for body text, `#99A9B5` (body-light) for secondary text
- Pill-shaped buttons (rounded-full) with burgundy fill and lime-green hover transition
- Decorative underline bars beneath section headings (`.heading-underline` class in global.css)
- Dark burgundy gradient hero sections with white text
- Generous whitespace and clean sans-serif typography
